Easy Email Setup with EmailJS (No Server Required)

Step 1: Sign up for EmailJS

Go to EmailJS.com and create a free account

Step 2: Add Email Service

In EmailJS dashboard, add Gmail as your email service

Step 3: Create Email Template

Create a template with these variables:

From: {{from_name}} ({{from_email}})
Subject: {{subject}}
Message: {{message}}

Step 4: Get Your IDs

Copy your Service ID, Template ID, and Public Key from EmailJS dashboard

Step 5: Use This Code

Replace the contact form JavaScript with this EmailJS version:

// Add EmailJS script to your HTML head:
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@3/dist/email.min.js"></script>

// Initialize EmailJS
emailjs.init("YOUR_PUBLIC_KEY");

// Contact form function
function initContactForm() {
  const contactForm = document.getElementById('contactForm');
  if (contactForm) {
    contactForm.addEventListener('submit', function(e) {
      e.preventDefault();
      emailjs.sendForm('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', this)
        .then(() => alert('Message sent successfully!'))
        .catch(() => alert('Failed to send message'));
    });
  }
}

Benefits of EmailJS:

  • No server-side code needed
  • No app passwords required
  • Works with static hosting
  • Free tier available
